# @feature-hub/module-loader

[![Package Version][package-badge]][package-npm]
[![Website][website-badge]][website]

A universal module loader.

## Installation

### Using Yarn

```sh
yarn add @feature-hub/module-loader
```

### Using NPM

```sh
npm install @feature-hub/module-loader
```
