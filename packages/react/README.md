# @feature-hub/react

[![Package Version][package-badge]][package-npm]
[![Website][website-badge]][website] [![API][api-badge]][api]

A Feature Hub integrator for React.

## Installation

### Using Yarn

```sh
yarn add @feature-hub/react
```

### Using NPM

```sh
npm install @feature-hub/react
```
